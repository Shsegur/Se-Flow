# Se-Flow

Web app React + Vite para Se-Flow.

## Ejecutar local

```bash
npm install
npm run dev
```

## Build producción

```bash
npm run build
```

## Firebase

Este proyecto ya incluye placeholders para:

- `__firebase_config`
- `__app_id`
- `__initial_auth_token`

Si querés conectar Firebase real, actualizamos `vite.config.js` con tus valores.
